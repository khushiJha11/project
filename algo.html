Disk Scheduling Algorithm

Let's now more about disk scheduling algorithm

First Come First Serve:-


1.In FCFS the disk controller chooses the next request in the queue to service.



View Algorithm
Shortest Seek Time First:-

1.SSTF selects the request with the least seek time from the current head position. SSTF chooses the pending request closest to the current head position.



View Algorithm
SCAN:-

1.The disk arm starts at one end of the disk and moves towards other end, servicing requests as it reaches each cylinder. At the other end, the direction of head movement is reversed, and servicing continues.


View Algorithm
LOOK:-

1.Same as SCAN but with the difference that the arm goes only as far as the final request in each direction. Then, it reverses direction immediately without going all the way to the end of the disk.
